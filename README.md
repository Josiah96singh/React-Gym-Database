# Gym Database App

Gym customer database made with React.

# Functionalities

1. Detailed Customer table(Pagination, Add, Update, Delete Customers)
2. Customer Trainings (Delete, Add)
3. Calendar View showcasing customer training schedule.

# Technology Stacks

1. React
2. React-Router
3. Moment js
